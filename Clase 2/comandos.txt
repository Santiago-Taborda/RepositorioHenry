git init
inicializar un repositorio local

git status

git add ''
agregar al commit solo el archivo nombrado

git add .
agregar al commit todos los archivos dentro de la carpeta

git commit -m ''
crear el commit y le agrega una descripcion

git log
mostrar los detalles del commit realizado

git push
subir los cambios deseados del commit a GitHub

git pull
recibir cambios de repositorio remoto a local
